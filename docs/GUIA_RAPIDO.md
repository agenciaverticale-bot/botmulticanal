# ⚡ Guia Rápido: 5 Minutos para Começar

Se você quer começar **agora mesmo** sem ler tudo, siga este guia rápido!

---

## 1️⃣ Preparar o Banco de Dados (1 minuto)

1. Vá para [https://supabase.com/dashboard](https://supabase.com/dashboard)
2. Selecione seu projeto
3. Clique em **"SQL Editor"** → **"New Query"**
4. Abra o arquivo: `docs/SETUP_DATABASE.sql`
5. Cole TODO o conteúdo
6. Clique em **"Run"**
7. Você deve ver: **"Success"** em verde

✅ Pronto!

---

## 2️⃣ Instalar Dependências (1 minuto)

No terminal, na pasta do projeto:

```bash
cd d:/programacao/botmulticanal
pnpm install
```

✅ Pronto!

---

## 3️⃣ Iniciar o Servidor (1 minuto)

```bash
pnpm dev
```

Abra: `http://localhost:3000`

✅ Pronto!

---

## 4️⃣ Criar Sua Primeira Regra de Chatbot (1 minuto)

1. Clique em **"Configurações"**
2. Vá para aba **"Chatbot"**
3. Preencha:
   - **Nome**: Saudação
   - **Palavras-chave**: olá, oi
   - **Resposta**: Olá! Como posso ajudar?
   - **Plataforma**: both
4. Clique em **"Criar Regra"**

✅ Pronto!

---

## 5️⃣ Ativar Notificações (1 minuto)

1. Vá para aba **"Notificações"**
2. Ative **"Notificações por E-mail"**
3. Defina limite para `10`
4. Clique em **"Salvar Configurações"**

✅ Pronto!

---

## 🎉 Você Terminou!

Seu chatbot está funcionando! Agora:

- 📱 Envie uma mensagem no WhatsApp/Instagram
- 💬 Você receberá a resposta automática
- 📊 Veja tudo no dashboard
